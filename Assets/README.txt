# How to Create New Items

1. Import the .fbx in Assets/Models/Items
2. Run the Tools > Import Tools > Configure Import Settings
3. Drag & Drop the model in any scene
4. Add the ItemMaker script
5. Select the Category
6. Click "Make Item"
7. Open the Icon Creation Scene, drag the newly created Prefab
8. Create the Icon
9. Run the Tools > Import Tools > Import Generated Icons
10. Browse the Assets/Textures/Items and place the Sprite in the correct category
11. Run the Tools > Asset Tools > Find Missing Sprites
12. Run the Tools > Asset Tools > Find Missing Prefabs
